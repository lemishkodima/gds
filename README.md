# GDC Implants — Богуслав

Односторінковий сайт центру імплантації та 3D-діагностики. Інтерфейс працює на React/Vite, а заявки надсилаються в Telegram через серверну функцію Vercel.

## Локальний запуск

```bash
npm install
npm run dev
```

Для локальної перевірки інтерфейсу Telegram не потрібен. Для повної перевірки API використовуйте `vercel dev` і створіть `.env.local` за зразком `.env.example`.

## Налаштування Telegram

1. Створіть бота через `@BotFather`.
2. Додайте бота в потрібний чат або групу.
3. Дізнайтеся ID чату (для групи він зазвичай починається з `-100`).
4. У Vercel → Project Settings → Environment Variables додайте:
   - `TELEGRAM_BOT_TOKEN`
   - `TELEGRAM_CHAT_ID`
   - `TELEGRAM_MESSAGE_THREAD_ID` — лише якщо потрібна конкретна тема форум-групи.
5. Перезапустіть deployment після додавання змінних.

Секрети не можна додавати у фронтенд-код або комітити в репозиторій.

## Перевірка перед публікацією

```bash
npm run check
npm run build
```

Після імпорту репозиторію Vercel автоматично виконає `npm run build` і опублікує каталог `dist`. Серверна функція форми доступна за адресою `/api/lead`.
